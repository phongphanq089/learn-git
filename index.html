git init : tạo trạng thái repository git status : kiểm tra trạng thái của dự án
git add : chuẩn bị các forder , các fou ở trạng thái chuẩn bị lưu
git reset : cho các fou trở về trạng thái ban đầu
git commit - m "" : lưu lại trạng thái các fou
git log, git log --oneline : kiểm tra lại history các fou, câu lệnh 2 ngắn hơn
git checkout (id) : đi đến commit mình muốn đến, muốn trở lại chính branch hiện tại thì => git checkout branch-name git checkout - b {branch name} : tạo ra 1 branch mới
git merge {name branch muốn hợp lại}
git branch - d {branch name} : xoá branch bạn muốn xoá
git push -u origin {name branch}
lấy branch từ git xuống : git fetch origin => git checkout -b {tên branch muốn lấy} origin/{branch}}}
git pull : lấy các sự thay đổi từ branch khác
